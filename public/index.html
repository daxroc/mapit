MapIt

MapIt is a tiny web application that shows various pre-defined locations ( latitude and longitudes ) on an Open Street Map while allowing users to specify and record custom locations in real time as favourite locations. This is acomplished with local storage, so that these favourites can be retrievied and displayed on return visits.

This application makes extensive use of Leaflet >insert link< map api in conjunction with OpenStreet map tiles, and uses jQuery for UI components and interaction with the DOM. The layout is controlled by Bootstrap.
