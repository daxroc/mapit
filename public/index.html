MapIt

MapIt is a tiny web application that shows pre-defined latitude and longitudes on an Open Street Map. It also allows users to add and record new locations in real time as their own favorite locations.

The storage element of this application is achieved using local-storage. This allows a returning user to maintain their favorites.

This application makes extensive use of Leaflet >insert link< map api in conjunction with OpenStreet map tiles. It also uses jQuery for UI components and event based interaction with the DOM. The layout is controlled by Bootstrap, which provides a great cross browser framework for a good responsive UX.
